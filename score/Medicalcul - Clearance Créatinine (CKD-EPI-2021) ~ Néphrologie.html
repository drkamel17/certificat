Clearance : ml/mn

Interprétation:
     Valeurs usuelles: Homme = 120 ml/mn (+/-20ml/mn) ; Femme = 95 ml/mn (+/-20ml/mn).
     L'équation du CKD-EPI (Chronic Kidney Disease - Epidemiology Collaboration) est plus précise que celle du MDRD et devrait remplacer cette dernière en usage clinique de routine. Elle est plus correcte pour les valeurs de clairance de la créatinine dépassant les 60 ml/min/1,73m2.
     La formule de 2021 ne prends plus en compte le contexte racial. Elle peut aussi ajouter en précision avec la Cystatine C. La formule Cystatine C seule est celle de 2012.
     Pour ne pas utiliser une des valeurs, il suffit de ne pas la remplir, mais au moins une des deux doit être renseignée.

Références:
     Formules:
       Créatinine seule : ClCr = 142 * min(Créat / k, 1) α * max(Créat / k, 1) -1.200* 0.9938 Age * (1.012 si sexe féminin) ; avec k=0.7 pour les femmes et 0.9 pour les hommes, et α=-0.241 pour les femmes et -0.302 pour les hommes.
       Créatinine + Cystatine C : ClCrCy = 135 * min(Créat / k, 1) α * max(Créat / k, 1) -0.544 * min(Cyst / 0.8, 1) -0.323 * max(Cyst / 0.8, 1) -0.778* 0.9961 Age * (0.963 si sexe féminin) ; avec k=0.7 pour les femmes et 0.9 pour les hommes, et α=-0.219 pour les femmes et -0.144 pour les hommes.
       Cystatine C seule : ClCy = 133 * min(Cyst / 0.8, 1) -0.499 * max(Cyst / 0.8, 1) -1.328 * 0.996 Age * (0.932 si sexe féminin).
